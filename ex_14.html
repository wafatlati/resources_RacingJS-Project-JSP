In the first white box area, you must be able to write BBcode from the input field. The second white box area display in real-time the text with the corresponding formating.


BBcode is structured as : [tag]text[/tag]

You must handle these tags :

  • B for Bold.
  • U for Underline.
  • S for Strikethrough.
  • COLOR=<color> to specify a color.
  • LINK=<url> display the text but clicking it redirect to the URL
